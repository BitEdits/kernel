SYNOPSIS

Synrc RT is an ANSI C99 POSIX RTOS for Synrc SKYNET application, AtomVM and Sokhatsky Commander. It supports a limited curated set of hardware platforms and plain make buildsystem. The size of zipped source distribution is limited to 2.88MB (~200K LOC).

HAL

Hardware Abstraction Layer

HAL defines set of include files for CHIPS and BOARDS supported by RTS.

CHIPS

CHIPS folder fefines hardware abstractions for CPU or SoC which include GPIO, I2C, SPI, UART and Timer harware definition for SoC built-ins.

ARMV7-A — ARMv7-A 32-bit Profile.
ARMV8-M — ARMv8-M 32-bit MCU Profile.
BCM2711 — Broadcom BCM2711.

BOARDS

BOARDS folder fefines hardware abstractions for I/O Boards where SoC is placed. Thus the BOARD primitives include by inheritance all CHIP primitives plus Boot, PWM, ADC, Sensors and Power hardware board facilities.

CXD56XX — Sony Spresense.
PICO2 — Raspberry Pi Pico 2.
VIRTIO — QEMU Virtio.
SIM — Simulation.
